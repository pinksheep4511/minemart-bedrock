Minemart Bedrock is a full-featured economy system for Minecraft Bedrock Edition. This addon is inspired by Minemart for Java Edition, except Minemart Bedrock uses NPCs instead of villagers and virtual shops with fixed prices.

This addon includes server shops, player shops, auctions, banking, loans, and a dynamic NPC role system.

## ✨ Features

- 🏬 Server Shops
- 🛍️ Player Shops
- 🏦 Banking System
- 📈 Credit Score Tracking
- 🏷️ Auction House
- 🤖 Role-Based NPC System
- ⚙️ Operator Configuration Menus
- 💾 Persistent Dynamic Property Storage
- 🔒 Economy Restriction Controls

## 🧠 NPC Roles

- Server Shop
- Banker
- Auction Master

## 💰 Economy Systems

- Server Shops (Buy & Sell Offers)
- Player Shops (Buy & Sell modes)
- Player-to-player Pay System
- Savings & Chequing Accounts
- Interest Accrual

## 🔧 Installation

1. Download the latest release.
2. Import into Minecraft Bedrock.
3. Enable in your world.
4. Ensure experimental features are enabled.
